# Context Switch Tracker

A minimal, high-performance React application designed to help users track and improve their focus by monitoring context switching.

## Features

- **Session Timer**: Precise tracking of work session duration.
- **Context Switch Detection**: Automatically detects when you leave the tab and increments a counter upon return.
- **Audio Feedback**: Subtle audio cue when a context switch is recorded.
- **Session Summaries**: detailed breakdown of time and interruptions after each session.

## Tech Stack

- **Framework**: Vite + React
- **Styling**: Tailwind CSS (Modern Dark Theme)
- **Icons**: Lucide React

## Development

```bash
npm install
npm run dev
```

Server will be available at:
- Local: http://localhost:8080
- Network: http://0.0.0.0:8080
